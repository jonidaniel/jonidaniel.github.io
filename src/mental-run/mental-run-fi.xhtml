PELIPORTAALI PITCHI IN ENGLISH

Mielenterveysongelmat ovat totta eikä nuorten henkinen hyvinvointi ole itsestäänselvyys. Mielialahäiriöt, ahdistuneisuushäiriöt, syömishäiriöt, psykoosit... Vaikka välillä on vaikeaa, muista silti tämä: selviät mistä tahansa. Pelastusta odotellessä kiihdytä vauhtiin ja käynnistä Mental Run!
Mental Run on 2D top-down juoksupeli Android-laitteille, jossa ohjaat arjen valintojen kanssa kamppailevaa nuorta läpi raskaan koulupäivän. Kerää terveydelle hyödyllisiä asioita matkan varrella ja ansaitse pisteitä. Hio näppäilynopeutesi ja reaktiokykysi äärimmilleen ja ansaitse paikkasi parhaiden joukossa!
